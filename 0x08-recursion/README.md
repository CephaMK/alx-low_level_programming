DIRECTOR

Contains files on ALX project tasks on the concept of recursion on C.

HEADER FILE: main.h
HELPER FILE: _putchar.c
TEST FILES: main.c files.

File 0-puts_recursion
C function that prints a string, followed by a new line.

File 2-strlen_recursion.c
C function that returns the length of a string.

File 3-factorial.c
C function that returns the factorial of a given number.

File 4-pow_recursion.c
C function that returns the value of x raised to the power of y.

File 5-sqrt_recursion.c
C function that returns the natural square root of a number.

File 6-is_prime_number.c
C function that returns 1 if the input integer is a prime number, otherwise returns 0.
