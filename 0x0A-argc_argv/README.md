DIRECTORY 0x0A-argc_argv

Directory has files on ALX projects tasks regarding the concept argc,argv in C Programming.

File 0-whatsmyname.c
C program that prits its name followed by a new line.

File 1-args.c
C program that prints the number of arguments passed into it followed by a new line.

File 2-args.c
C program that prints all arguments it receives.
	>All arguments should be printed, including the first one.
	>Only print one argument per line, ending with a new line.

File 3-mul.c
C program that multiplies two numbers.
	>program should print the result of the multiplication, followed by a new line.
	>program assumes that the two numbers and result of the multiplication can be stored in an integer.
	>If the program does not receive two arguments, your program should print Error, followed by a new line, 
	 and return 1.

File 4-add.c
C program that adds positive numbers.
	>Prints the result, followed by a new line.
	>If no number is passed to the program, print 0, followed by a new line.
	>If one of the number contains symbols that are not digits, print Error, followed by a new line, and return 1.
	>Program assumes that numbers and the addition of all the numbers can be stored in an int.
